Detailed Description

template<typename T>
class HyProblem< T >

Defines a physical problem.

The class HyProblem represents a physical problem described by a domain, its boundary, the corresponding PDE and the boundary conditions. The numerical solution is obtained by solving the linear system AX = F+G. A represents the matrix (determined by the equation of the problem), X is the unknown vector (the approximate solution), F
