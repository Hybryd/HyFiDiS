Parameters:
DISPLAY_MESH_GRID_STYLE If the value is 1, displays the mesh like a grid in the terminal, with dots at each node.
DISPLAY_WHOLE_MESH If the value is 1, displays the whole mesh (as a set of points). If 0, it only displays the domain defined by the HyBoundary. It avoids displaying useless points outside the interesting domain.
HY_EPSILON Arbitrary precision, used in the code of this library only.
HY_PI Approximation of the most beautiful number of all the universe.
QUIET If the value is 1, does not display any additionnal information like details of the problem or of the computation.
